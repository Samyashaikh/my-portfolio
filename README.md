# 💼 Developer Portfolio

A clean, dark-themed personal portfolio website for developers.

## 🚀 Features
- Fixed glassmorphism navbar
- Animated hero with typing effect
- About, Projects, Skills, Contact sections
- Smooth scroll reveal animations
- Fully responsive

## 🛠 Tech Used
- HTML5, CSS3, Vanilla JavaScript
- Google Fonts (Space Grotesk + Space Mono)

## ✏️ How to Customize
1. Replace `Your Name` with your actual name in `index.html`
2. Add your photo in `.img-placeholder` div
3. Update projects, skills, and contact info
4. Replace `resume.pdf` with your actual resume file

## 📁 File Structure
```
portfolio/
├── index.html
├── style.css
├── script.js
└── README.md
```
