## 🛠 Deployment & Hosting Setup
To host this portfolio page completely for free directly on GitHub Pages:
1. Create a new GitHub Repository named exactly `yourusername.github.io` (replace `yourusername` with your active account handle).
2. Upload these 4 files (`index.html`, `style.css`, `script.js`, `README.md`) straight into the root directory.
3. Upload your resume file and name it exactly `SAMYA-SHAIKH-21.pdf` into the exact same folder location so the download button hooks to it seamlessly.
4. Go to **Settings** -> **Pages** -> Ensure build source is set to **Deploy from a branch** (Main/Master -> `/root`) -> Click Save.
